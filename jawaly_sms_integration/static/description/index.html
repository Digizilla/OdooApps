How to send SMS

  1. From dashboard choose contacts
  2. After that , if you want to send sms to one contact Choose the contact
  3. After that click to action and choose send sms text message
  4. After click send sms text message, write the message and send it from
  5. If you want to send a message to more than one personFrom list view after that select the contacts you want to send the message to and from action choose send sms text message.
  6. After that if you want to send a message, write a message and click send now . If you want to write the message ,and then you will choose when to send it choose put in queue
  7. After that go to settings and choose Technical and form it choose phone/sms and click sms ,AFTER clicking , the messages will appear in the queue , and then you will send them from send now
  8. ALSO YOU CAN SEND A MESSAGE FROM MODULE SALES, THEN create a quotation and choose a customer and confirm after that click send sms
  9. From module jawaly, Click jawaly logs Show all the details of the message , including the creation date , message number , and content of the message.
